# PRODIGY_WD_03
 The Tic-Tac-Toe Web Application is an interactive game built with HTML, CSS, and JavaScript. Users can play the classic tic-tac-toe game, where the objective is to align three markers (X or O) in a row, column, or diagonal to win. The app is designed to handle user clicks, manage game state, and check for winning conditions, creating an engaging experience for players.
